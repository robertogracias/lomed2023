Single Invoice for Multiple Sale Orders

You can use this application to combine several sales orders from the same customer into a single invoice.

Key Highlights

  • Using this app user can create single invoice for multiple sale orders.
  • After creating invoice user can see that invoice from sale order also.
  • One combined invoice can be issued if you create numerous sales orders for the same customer. Invoices are typically not needed on a daily basis by most frequent customers, who instead look for them at regular intervals like once a week or once a month, etc.

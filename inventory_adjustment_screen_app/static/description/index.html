Description

Product

Here product is category wise.


Inventory Type Is One Product Category

When we select this option then we can select category and onclick start inventory then selected category's product set in line.




Onclick validate inventory then adjustment show in inventory adjustment tab.


Inventory Type Is All Product

When we select this option and onclick start inventory then all storable product set in line.


Inventory Type Is Ony One Product

When we select this option then we can select product and onclick start inventory then selected product set in line.


Inventory Type Is Select Product Manually

When we select this option then we can select more than one product and onclick start inventory then selected product set in line.


Inventory Type Is One Lot/Serial Number

When we select this option then we can select lot and onclick start inventory then selected lot's product set in line.
